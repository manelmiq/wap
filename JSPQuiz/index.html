1. What is the main value of Java Server Pages?
The main proposal it's separate the view from the business logic.
2. How are JSP pages related to servlets?
The JSP page is a servlet page. But first it's need to be compiled and turn in a servlet. Also the JSP respect the life cycle of servlet.
3. How are JSP pages related to HTML?
Because you can insert java code inside a html using EL element. Later you can put some tags in order to increase the power.
